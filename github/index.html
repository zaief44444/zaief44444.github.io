最近のgithubはクラウドビルド環境などが素晴らしいためその機能も使いこなしたいと思う. gitは操作対象のブランチを省略できるので混乱しやすい. 【Git】基本コマンド を丸写しにした部分が多い. 


git init
git add *
git commit -m "initial commit"

git push origin master //master 更新. 



git add . //すべてのファイル・ディレクトリ
git add *.css //すべてのCSSファイル
git add -n //追加されるファイルを調べる
git add -u //変更されたファイルを追加する
git rm --cached //addしてしまったファイルを除外



git commit -a //変更のあったファイルすべて
git commit --amend //直前のコミットを取り消す
git commit -v //変更点を表示してコミット

fzf解説では git add をfzfを使って変更のプレビューをみることができ, とても便利.
