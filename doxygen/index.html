doxygen解説

Mar 12, 2020

プラグラムの規模が大きくなると依存関係などが煩雑になる. よりプログラムの見通しをよくするためにこのツールを使ってみる.

インストール


brew install doxygen
brew install graphviz

設定

Doxyfileを生成し, 設定をする. まずソースファイルがあるディレクトリへ行き次のコマンドする. 


doxygen -g

Doxyfileを編集する. 


HAVE_DOT = YES
EXTRACT_ALL = YES
EXTRACT_PRIVATE = YES
EXTRACT_STATIC = YES
CALL_GRAPH = YES

次のコマンドでhtml, latexが生成される 


doxygen Doxyfile
