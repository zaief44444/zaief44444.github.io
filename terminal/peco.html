peco解説

July 17, 2020

pecoはzshの検索機能ツールである. 検索というのはコンピューター操作において最も基本的な操作であり, ブラウザの存在が検索の重要性を事実たらしめる. 当然, ターミナル上でも検索行為は可能だ. brew install peco でインストールできる. .zshrcに次を記述する. 


function peco-history-selection() {
    BUFFER=`history -n 1 | tail -r  | awk '!a[$0]++' | peco`
    CURSOR=$#BUFFER
    zle reset-prompt
}

zle -N peco-history-selection
bindkey '^R' peco-history-selection
